# Personal Projects

This repository serves as a collection of personal projects focused on data analysis, business problem solving, and cloud engineering. It is organized to document practical work in a structured and professional manner, with room for future expansion into more advanced cloud-based implementations.

## Overview

The workspace currently contains projects in two main areas:

- Finance and customer-focused data projects
- Personal AWS cloud projects

## Repository Structure

### Finance and Customer-Focused Projects
This section includes analytical and data preparation work related to business performance and customer behavior.

- 01_Company_profit_expenditure
  - Contains an analysis notebook focused on company profit and expenditure data.

- 02_Customer Churn Project
  - Includes project documentation and an ETL/data preparation notebook for customer churn analysis.

### AWS Projects
This section is currently planned for personal AWS development work.

- The AWS projects folder will be used to build and document AWS-based solutions from the ground up.
- Initial work will focus on simple YAML-based Infrastructure as Code templates and CloudFormation examples.
- As experience grows, this area will expand to include more advanced AWS architectures and deployment scenarios.
